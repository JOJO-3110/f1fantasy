F1 Fantasy Setup

Einmalige Einrichtung — danach können alle Spieler die Website gemeinsam nutzen.

1 Supabase Account erstellen

Gehe auf supabase.com → "Start your project" → kostenlosen Account erstellen.

2 Neues Projekt erstellen

Klick auf "New project" → Name: f1fantasy → Region: EU West → Passwort dir merken → "Create new project".

3 Datenbank-Tabelle anlegen

Im Supabase Dashboard → links auf "SQL Editor" → "New query" → folgenden Code einfügen und auf "Run" klicken:

CREATE TABLE liga_state ( key TEXT PRIMARY KEY, value JSONB, updated_at TIMESTAMPTZ DEFAULT now() ); ALTER TABLE liga_state ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow all" ON liga_state FOR ALL USING (true) WITH CHECK (true);
